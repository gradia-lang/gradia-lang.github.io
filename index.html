About

Gradia name's origin is from "Gradual typing". So, Gradia adopt gradual way as type system. It improves trustablity
Implemented by Rust, and it splited core interpreter logic and user-interface. So, Gradia is run on browser using WASM too!
Have a happy hacking of fantatic Gradia programming world!
