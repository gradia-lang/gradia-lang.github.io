About

Gradia name's origin is from "Gradual typing". So, Gradia adopts gradual way as type system. It improves trustability
Implemented by Rust, and it's splited core interpreter logic and user-interface. So, Gradia is run on browser using WASM too!
Have a happy hacking of fantastic Gradia programming world!
